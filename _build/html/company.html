Company¶

Endpoint: GET v1/{:PARTNER_SLUG}/companies

Descrição: Essa chamada retorna todas as companies (restaurantes) cadastrados na Onyo e o seu respectivo token. Esse token deverá ser utilizado em todas as chamadas subsequentes.

Essa chamada recebe autenticação via JWT, com o token do tipo Bearer recebido na chamada de login acima.

Headers:

PARÂMETRO

TIPO

OBRIGATÓRIO

COMENTÁRIO

Authorization

String

Sim

Deve conter o token Bearer de usuário retornado pelo login.

Exemplo de URL (com layout simples e sem paginação): v1/company
